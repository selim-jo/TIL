# list 초기화 시 null vs new 사용 차이
- list 초기화 시 null 또는 new를 사용하는데 이 차이를 정확히 몰라 인터넷에 서치해보며 찾아 정리하였다.
- null: 직접 값 제어
      : 아무것도 없는 상태, 메모리 주소x
       (초기화 후 값 넣고 사용, 나중에 null로 초기화 가능)
- new: add, put, getter, setter 메서드 사용한 값 제어
     : 객체를 생성하여 메모리 주소 부여
       (초기화 후 바로 사용, 이미 초기화 상태이므로 이후엔 add로만 데이터 추가 가능)
