# ModelAndView vs String 차이
- controller 구현을 하면서 어떤 곳에서는 return 타입을 ModelAndView로 하고 어떤 곳에서는 String을 사용한다.
  그래서 이에 대한 차이점에 대해 정리하였다.
- ModelAndView를 리턴할 경우에는 ModelAndView 클래스의 생성자나 setViewName() 메서드를 이용해 뷰 이름을 지정할 수 있다.
- String을 리턴할 경우애는 문자열 값이 뷰 이름으로 사용된다.
