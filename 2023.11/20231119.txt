# 자바 웹 기술 5장-6장 정리 및 실습
- 자바를 사용하여 웹 개발을 하면서 한번 개념을 정리할 필요가 있다 생각이 들어 관련 내용을 노션에 정리하였다.
- 자세한 정리는 노션에 하였고, 여기에는 정리 및 실습을 하면서 잘못 알고 있었던 개념을 한번 더 복기하였다.

// 서블릿 동작 과정
1) 고객1이 http://localhost:8080/test로 요청
2) 톰캣은 해당 서블릿이 메모리에 로드되어 있는지 확인
3) init() 메서드를 호출하여 해당 서블릿 인스턴스를 메모리에 로드
4) doGet()이나 doPost() 메서드 호출하여 서비스
5) 고객2가 http://localhost:8080/test 동일한 요청
6) 톰캣은 다시 해당 서블릿이 메모리에 로드되어 있는지 확인
7) 메모리에 있는 것이 확인되므로 init() 메서드를 호출하지 않고 바로 doGet()이나 doPost() 메서드 호출하여 서비스
따라서, 서블릿 같은 경우 동일한 요청이 들어오면 메모리에 존재하는 서블릿을 재사용하므로 빠르고 효율적으로 동작시킬 수 있음
