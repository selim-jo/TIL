# prj 하면서 겪은 오류 해결법 정리
- Eclipse에서는 정상적으로 실행되던 프로젝트가 Intellij에서는 오류가 발생하였다.
  거의 반나절 동안 오류를 찾느라 꽤 고생을...
  But, 오류는 생각보다 간단한 거였다.
  Eclipse에서는 특정 패키지 아래에 즉, /main/java/ 아래에 mapper 관련 xml파일을 같이 두었는데 이게 문제였다.
  Intellij에서는 java 디렉토리 아래에 있는 xml 파일을 인지하지 못하고, maven 빌드시에도 target으로 복사되지 않았다.
  그래서 mapper 관련 xml 파일을 /main/resources/ 아래로 옮기고 다시 실행하니 정상적으로 작동하였다.
  두 개발환경도구의 차이에 대해 자세히 모르다보니 이런 오류를 겪은 것 같다...
  시간이 될 때 두 개발환경도구의 차이에 대해 정리하고 IntelliJ에서는 /main/java/ 아래에 xml파일을 못불러오는지 방법을 찾아봐야겠다.
